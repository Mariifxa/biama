Projeto

O BiAMa é um projeto cientifico desenvolvido por professores e alunos, com o intuito de criar uma biblioteca
física e digital direcionada ao estudo de diversos tipos de materiais. Esta biblioteca está disponível a qualquer
pessoa que esteje interessada, assim como também podem contribuir para o alargamento e melhoria da mesma. Esta biblioteca
serve como ferramente para a pesquisa eficaz de vários tipos de materiais. Mesmo distante da biblioteca física, através
deste website, o utilizador poderá ter sempre acesso à base de dados de materiais. Caso ganhe interesse por esta iniciativa,
pode se dirigir à Biblioteca de Materiais que estiver mais próxima de si, pelo que pretendemos que no futuro existam
muitas mais disponíveis.
